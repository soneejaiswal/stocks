*********************************************** Stocks API Implementation for Payconiq ********************************************************

Stock APIs : 
1. GET /api/stocks (get a list of stocks)
2. GET /api/stocks/{id} (get one stock from the list) 
3. PUT /api/stocks/{id} (update the price of a single stock)
4. POST /api/stocks (create a stock)
5. DELETE /api/stocks/{id} (delete a stock)


Technology stack used : 
1. Java 8
2. Spring Framework & Spring boot 
3. Maven
4. angular 
5. Junit

How to run :
1. run command  'mvn spring-boot:run' from the commandLine by doing cd to /pathToProject
 
JavaDoc/Documentation : 
1. Swagger - can be accessed with endpoint 'http://localhost:8080/swagger-ui.html', after running the project

Test cases :
1. Junit test cases in src/test/java
